# Course Review Helper SKILL 📚

智能课程复习助手 - 全学科通用的TRAE IDE技能，支持多格式资料混合输入，自动生成Obsidian兼容的复习笔记。

## 🤔 为什么会有这个项目？

### 开发背景
这个技能诞生于期末周的焦虑时刻。

每当期末复习时，我总是让AI重复生成类似的复习笔记提示词：
- "帮我总结第一章PPT"
- "提取这个PPT的关键知识点"
- "生成一些练习题"
- "整理公式推导过程"

一遍又一遍地输入这些提示词实在太浪费时间了，索性把这些重复性的指令封装成一个TRAE SKILL，让AI自动按照固定的流程帮我生成复习笔记。

### 实际定位
**⚠️ 诚实说明**：
- 这不是一个完美、功能强大的工具
- 使用体验可能不够流畅
- 生成的笔记质量取决于AI的理解能力
- 可能需要你手动调整一些内容

### 核心价值
虽然功能不够完善，但对我个人来说，最大的价值是：
- **节省重复输入时间**：不用每次都手写长篇提示词
- **Obsidian笔记更易复习**：用Obsidian的双向链接、标签系统、折叠答案来复习，比单纯翻看PPT方便很多
- **结构化复习资料**：虽然本质上还是在看PPT内容，但有了知识点清单、考点表格、自测题库，复习更有方向性

如果你也遇到类似的问题，希望这个小工具能帮到你。如果觉得不好用，欢迎提出改进建议！

## ✨ 功能特性

### 🎯 核心能力
- **多格式支持**：PPT、PDF、图片（PNG/JPG）、Word文档、纯文本/Markdown
- **智能识别**：自动识别课程类型（数学/理科、工科、文科、语言类、实验类）
- **范围选择**：提供三种复习范围选项，灵活适应不同复习场景
- **Obsidian优化**：生成的笔记完全兼容Obsidian，支持双向链接、标签系统、折叠答案

### 📋 输出内容
每个章节自动生成：
1. **知识点总结**：核心概念、公式推导、逻辑框架
2. **高频必考点清单**：考察频率、常见题型、记忆要点
3. **分题型自测题库**：填空、名词解释、计算/推导、简答/论述、翻译/写作

### 🔄 工作流程
```
扫描文件 → 汇报清单 → 选择范围（A/B/C）→ 生成笔记
```

## 🚀 快速开始

### 安装方法

#### 方法一：手动安装（推荐）
1. 下载此仓库的 `SKILL.md` 文件
2. 复制到你的TRAE技能目录：
   ```
   Windows: C:\Users\<你的用户名>\.trae-cn\skills\course-review-helper\
   macOS/Linux: ~/.trae-cn/skills/course-review-helper/
   ```
   最终文件路径应为：
   ```
   Windows: C:\Users\<你的用户名>\.trae-cn\skills\course-review-helper\SKILL.md
   ```
3. 重启TRAE IDE或刷新技能列表

#### 方法二：在TRAE中直接使用
如果你不想手动下载，也可以在TRAE IDE中输入以下内容（但每次都要输入，比较麻烦）：
```
我现在需要复习【课程名称】，所有复习资料都在【文件夹路径】中。
请帮我按照以下流程生成复习笔记：
1. 先扫描文件夹，列出所有PPT、PDF、图片文件
2. 让我选择复习范围（A：仅PPT；B：指定文件；C：全部资料）
3. 按章节生成Obsidian笔记，包含知识点总结、考点清单、自测题库
4. 笔记格式要求：使用frontmatter、双向链接、折叠答案
```

**为什么推荐手动安装？**
手动安装后，你只需要一句话就能触发整个流程：
```
使用 course-review-helper 技能复习我的课程
```

### 使用前准备

#### 1. 组织复习资料
将所有复习资料放在一个文件夹中，建议按以下方式命名：
```
D:\复习资料\课程名称\
├── 第一章.pptx
├── 第一章_补充.pdf
├── 第二章.pptx
├── 期末总结.pptx
├── 试题集.pdf
└── 手写笔记.jpg
```

#### 2. 检查文件格式
确保你的复习资料是以下格式之一：
- PPT/PPTX（主要教学内容）
- PDF（教材扫描件、试题集、补充讲义）
- 图片（PNG/JPG）：手写笔记、课本拍照、板书照片
- Word文档：课程大纲、习题集
- 纯文本/Markdown：自行整理的笔记

### 使用示例

以复习"微波遥感"课程为例：

#### 第一步：激活技能
在TRAE IDE中输入：
```
使用 course-review-helper 技能，我的微波遥感复习资料在 D:\复习资料\微波遥感\
```

#### 第二步：扫描文件
技能会自动扫描文件夹，输出类似：
```
📂 已扫描文件夹，发现以下文件：

PPT文件（共5个）：
- 微波遥感_第一章.pptx ✓（正课内容）
- 微波遥感_第二章.pptx ✓（正课内容）
- 微波遥感_第三章.pptx ✓（正课内容）
- 微波遥感_期末总结.pptx ✓（总结类）
- 微波遥感_习题课.pptx ✓（习题类）

PDF文件（共2个）：
- 微波遥感教材.pdf ✓（教材扫描）
- 期末试题集.pdf ✓（试题）

图片文件（共3个）：
- 板书照片1.jpg ✓（板书）
- 手写笔记.jpg ✓（笔记）
- 公式推导.png ✓（公式）
```

#### 第三步：选择复习范围
技能会询问你选择复习范围：
```
请选择复习范围：
A. 仅以PPT内容为准（推荐期末复习）
B. 以指定文件为准（如只复习期末总结.pptx）
C. 综合使用所有资料（全面复习）

你的选择是？
```

**我的推荐**：
- **期末突击复习**：选A，以PPT为准，效率最高
- **老师划定特定资料**：选B，精准复习
- **需要全面覆盖**：选C，但需要人工判断内容冲突

#### 第四步：生成笔记
选择A后，技能会按章节生成Obsidian笔记：
- 第一章笔记（包含知识点、考点、题库）
- 第二章笔记
- 第三章笔记
- 期末总复习MOC（整合所有章节）

每份笔记会保存为独立的 `.md` 文件，可以直接用Obsidian打开。

## ⚠️ 使用限制与注意事项

### 可能遇到的问题

#### 1. AI理解偏差
AI可能会误解PPT内容，导致生成的知识点不准确。**建议**：
- 生成后对照原PPT快速浏览
- 发现错误及时在对话中纠正（如"第一章第3个知识点定义不对，应该是XXX")
- 重新生成或手动修改笔记

#### 2. 链接错误
Obsidian双向链接可能指向不存在的笔记。**建议**：
- 检查生成的链接是否正确
- 使用Obsidian的"链接修复"插件
- 或在对话中要求AI修复链接

#### 3. 图片识别不准确
图片中的公式、表格可能识别错误。**建议**：
- 关键公式和表格最好手动核对
- 对于复杂的公式推导，建议直接看原图片

#### 4. 生成速度慢
大量资料可能需要较长时间生成。**建议**：
- 先选择重要章节复习
- 分批生成（如只生成第一章和期末总结）
- 避免一次性处理太多文件

### 我的实际使用经验

#### 成功案例
- **工科课程（微波遥感）**：公式推导部分效果不错，AI能正确识别大部分公式
- **文科课程（马原）**：论述框架生成质量较高，适合考前突击背诵

#### 需要改进的地方
- **图片识别**：手写笔记识别准确率不高，建议优先使用打印资料
- **跨章节关联**：生成的链接有时不够准确，需要手动调整
- **题库质量**：生成的题目有时过于简单，建议结合真实试题

### 什么时候不适合使用？

- 课程资料过于复杂（如大量复杂的图表、特殊符号）
- 需要极高准确度的考试（建议还是自己整理）
- 资料格式混乱（多个版本的PPT、大量重复内容）
- 时间紧迫（生成笔记本身需要时间，不如直接看PPT）

### 什么时候适合使用？

- 期末突击复习，需要快速整理PPT内容
- 习惯用Obsidian复习，觉得比看PPT效率高
- 课程内容相对规整，PPT结构清晰
- 有时间对照生成的笔记进行修正
